# Clinical Response Patterns

## Common reviewer issues

- Overstated causality in observational evidence.
- Patient-outcome claims from surrogate or workflow endpoints.
- Missing protocol/SAP or registry consistency.
- Insufficient harms, missing data, or multiplicity handling.
- Weak external validation or calibration in AI/prediction studies.
- Limited generalizability to real-world clinical settings.

## Safe response moves

- Add endpoint definition and time horizon.
- Add absolute effect size and confidence interval.
- Add limitations language for unmeasured confounding or external validity.
- Add sensitivity analysis only when the user supplies results.
- Reframe unsupported benefit claims as hypothesis-generating or workflow-relevant.
