# Clinical Evidence Screening

## Claim support checks

- Match population, setting, intervention/exposure/test/model, comparator, and outcome.
- Distinguish clinical endpoints from surrogate or workflow metrics.
- For diagnostic/prognostic claims, check reference standard, calibration, discrimination, decision thresholds, and external validation.
- For AI claims, check training/validation split, external site validation, prospective evaluation, workflow integration, monitoring, bias, privacy, and safety.
- For treatment claims, check randomization, blinding, allocation concealment, follow-up, protocol/SAP, intention-to-treat, absolute effects, harms, and multiplicity.

## Reporting guideline routing

- Randomized trial: CONSORT; CONSORT-AI if AI intervention.
- Trial protocol: SPIRIT; SPIRIT-AI if AI intervention.
- Observational: STROBE.
- Diagnostic accuracy: STARD.
- Prediction model: TRIPOD+AI when regression or ML model is involved.
- Systematic review/meta-analysis: PRISMA.

## Output principle

Never imply that improved detection, workflow efficiency, or model accuracy proves mortality, morbidity, or patient-outcome benefit unless the source directly supports that endpoint.
