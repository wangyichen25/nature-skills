# Clinical Figure Patterns

## QA checklist

- Denominator visible.
- Endpoint definition and time horizon visible.
- Comparator or reference standard visible.
- Confidence interval or uncertainty visible when inferential.
- Missing data, censoring, and attrition shown when relevant.
- Absolute effects shown when clinical decisions depend on magnitude.
- Harms shown with the same seriousness as benefits.

## AI-specific checks

- Include external validation site or cohort.
- Show calibration, not only AUROC.
- Identify decision threshold and intended workflow.
- Show subgroup or fairness checks when population mix matters.
- Avoid implying deployment benefit from retrospective model performance alone.
